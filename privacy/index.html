3. How the Service Works

Understanding the technical implementation helps clarify our privacy stance:

Static Website: This is a completely static website hosted on GitHub Pages. There is no server-side code, no backend, and no database. Every shortlink is a simple HTML file with a meta refresh tag and JavaScript redirect.

When You Visit a Shortlink:

  1. Your browser requests the HTML file from GitHub's CDN
  2. The HTML contains a redirect to the destination URL
  3. Your browser immediately redirects you
  4. We never see or log this request
